## 目前仍然存在的问题（下次清洗需要解决的问题）

上述筛掉数据的第 3 个条件我们是通过总个数来筛选的，这样的话有的数据是筛不掉的，如：

* `筛【数)据`；虽然 `左开` 的符号和 `右开` 的符号总数相等，但是左开和右开的符号并不能匹配
* `筛）数（据`；总数相等，符号匹配，但是符号的位置不匹配（出现顺序不匹配）
* 其它与第 3 个筛选条件无关的问题（待发现）
